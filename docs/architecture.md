# Arquitectura Inicial

## Frontend

Aplicacion React con Vite. La estructura inicial separa paginas, componentes compartidos y estilos globales.

Paginas base:

- Login
- Registro
- Publicacion de servicio
- Listado de servicios

## Backend

API Node.js con Express. La estructura inicial separa la app, el servidor, rutas y controladores.

Rutas base:

- `/api/auth`
- `/api/services`

La logica de autenticacion, persistencia y reglas de negocio queda pendiente para futuras iteraciones.
